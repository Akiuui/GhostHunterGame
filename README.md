Ghost Hunter
A fast-paced 3D action game built with Unity. The project focuses on clean game logic, smooth character movement, and an engaging ghost-clearing mechanic.

🚀 Overview
Ghost Hunter challenges players to clear environments of supernatural entities within a set timeframe. This project was developed to explore Unity’s 3D engine capabilities, focusing on collision detection, state management, and user interface integration.

🛠️ Tech Stack
Engine: Unity 3D

Language: C#

Key Features:

Dynamic Ghost Counter system.

Timer-based gameplay loop.

Physics-based movement and interaction.

Custom UI for real-time player feedback.

🕹️ Gameplay Features
Ghost Detection: Logic-driven ghost tracking and elimination.

Time Trial: Integrated countdown system that adds a layer of difficulty.

Action Mechanics: Responsive controls designed for a seamless player experience.
